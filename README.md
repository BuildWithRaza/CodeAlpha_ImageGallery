📸 Project: Image Gallery using HTML, CSS & JavaScript

This project helped me strengthen my frontend development skills by building a responsive and interactive image gallery with category-based filtering.

🔹 HTML (Structure)

• Created the webpage layout and heading.
• Displayed multiple images in a gallery.
• Built a hidden popup (lightbox) to display enlarged images.
• Added category filter buttons
.
🎨 CSS (Styling)

• Designed a clean and responsive gallery layout.
• Added spacing and rounded corners for a modern look.
• Implemented a smooth zoom effect on image hover.
• Styled filter buttons for a better user experience.
• Created a dark overlay background for the popup.

💻 JavaScript (Functionality)

• Opened images in a popup when clicked.
• Added Next (>) and Previous (<) navigation buttons.
• Enabled closing the popup with a Close (×) button.
• Implemented category-based image filtering so users can quickly browse images by category.

🚀 Project Features

✅ Responsive image gallery layout.
✅ Filter images by category.
✅ Click any image to view it in a lightbox popup.
✅ Navigate through images using Next and Previous buttons.
✅ Close the popup with the Close (×) button.
✅ Smooth hover animations and modern UI design.
